# Fruits & Vegetables E-Commerce Website (Delicacy)

## Project Overview
This is a responsive e-commerce frontend website for a fruits and vegetables store called **Delicacy**. It allows users to browse products, view categories, explore featured items, and check product details through a modern and interactive UI.

## Features
- Responsive navigation bar with mobile menu
- Hero section with Swiper slider
- Product categories section
- Popular products grid with pricing and discounts
- About section describing the store
- Customer reviews section
- Newsletter subscription form
- Footer with links and social media icons
- Smooth responsive design for all devices

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Swiper.js (for sliders)  
- Boxicons (for icons)

## Purpose
This project was built to practice frontend development, responsive web design, and building a real-world e-commerce UI layout.
